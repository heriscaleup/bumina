# Produksi Freeze Dried Food dengan Standar Kualitas Terjaga

Landing page SEO untuk **Raja Freeze Dried Food** — topik: Produksi Freeze Dried.

## Struktur Proyek

```
.
├── index.html       # Halaman utama (single-page, tanpa build step)
├── favicon.svg       # Favicon SVG (warna brand)
├── robots.txt        # Aturan crawler
├── sitemap.xml        # Sitemap untuk SEO
├── vercel.json        # Konfigurasi deployment Vercel (headers, caching)
├── images/            # Gambar hero
└── .gitignore
```

## Menjalankan Secara Lokal

Situs statis tanpa build step, cukup buka `index.html` di browser atau jalankan:

```bash
npx serve .
```
