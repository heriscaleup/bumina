# Raja Cutting Laser — Landing Pages

10 standalone static landing page variants for **Raja Cutting Laser** (masjid/Islamic ornament laser cutting). Each `lp-XX` folder is a self-contained static site (index.html, images/, vercel.json) with no build step — see the `README.md` inside each folder for details.
